# 💰 Finance Tracker Backend (FastAPI)

## 📌 Overview
This project is a Python-based finance tracking backend system built using FastAPI.  
It allows users to manage financial transactions and analyze their spending patterns.

---

## 🚀 Features
- CRUD operations for transactions
- Filtering (type, category, date)
- Financial analytics (income, expenses, balance)
- Role-based access control (admin, analyst, viewer)
- SQLite database persistence
- REST API endpoints
- Input validation and error handling

---

## 🛠️ Tech Stack
- FastAPI
- SQLite
- SQLAlchemy
- Pydantic
